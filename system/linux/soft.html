    • redis.conf (opens new window)

    • 默认存储数据的文件：dump.rdb

    • docker run -p 6379:6379 --name redis -v /root/redis/redis.conf:/etc/redis/redis.conf -v /root/redis/data:/data -d redis:latest redis-server /etc/redis/redis.conf --appendonly yes (opens new window)

      redis-server /etc/redis/redis.conf：这个是关键配置，让redis不是无配置启动，而是按照这个redis.conf的配置启动 –appendonly yes：redis启动后数据持久化
