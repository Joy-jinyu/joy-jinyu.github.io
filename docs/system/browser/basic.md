## 获取dom元素高度和宽度

### body

   + 可见区域高：clientHeight
   + 可见区域宽：clientWidth
   + 可见区域高（包括边线的高）：offsetHeight
   + 可见区域的宽（包括边线的宽）：offsetWidth
   + 正文宽：scrollWidth
   + 正文高：scrollHeight
   + 滚动条（滑动标识）距离顶部：offsetTop
   + 滚动条（滑动标识）距离左侧：offsetLeft

### 元素

   + 实际高度：offsetHeight
   + 实际宽度：offsetWidth
   + 距离左侧：offsetLeft
   + 距离顶部：offsetTop
