# 一、Markdown语法

# 1. 标题

  • 语法

    # 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

  • 效果

    略

# 2. 字体

  • 语法

    **我是加粗**
*我是斜体*
***我是斜体加粗***
~~我是删除线~~

  • 效果

    我是加粗

    我是斜体

    我是斜体加粗

    我是删除线

# 3. 引用

  • 语法

    > 一层引用
>> 二层引用
>>> 三层引用
....
>>>>> 多层引用

  • 效果

    一层引用

    二层引用

    三层引用 ....

    多层引用

# 8. 表格

  • 语法

    |表头|表头|
|:-:|:-:|
|内容|内容|

第二行分割表头和内容（使用-）
文字默认居左
-两边加：表示文字居中
-右边加：表示文字居右
-左边加：表示文字居左

  • 效果

    表头 表头
    内容 内容
