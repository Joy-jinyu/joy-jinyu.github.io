    # PageSpeed Tools

    PageSpeed的分析基于一个分为五类的最佳实践列表：

    * 优化缓存——让你应用的数据和逻辑完全避免使用网络

    * 减少回应时间——减少一连串请求-响应周期的数量

    * 减小请求大小——减少上传大小

    * 减小有效负荷大小——减小响应、下载和缓存页面的大小

    * 优化浏览器渲染——改善浏览器的页面布局
